Obsidial-plugin, автоматически вставляющий заранее заданный шаблон в новосозданный файл.

Работает по следующему принципу: к папке прикрепляется шаблон (это может быть любой md-файл), после чего во все создаваемые в ней файлы будет копироваться содержимое шаблона.
